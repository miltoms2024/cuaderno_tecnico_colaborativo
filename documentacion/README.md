# Cuaderno Técnico Colaborativo

Proyecto final del programa Full Stack Development.  
La aplicación permite a los usuarios registrarse, iniciar sesión y crear notas con título, contenido y archivos adjuntos.

## Funcionalidades
- Registro e inicio de sesión.
- Crear, guardar y publicar notas.
- Adjuntar archivos a las notas.
- Editar y eliminar notas.
- Descargar notas o archivos adjuntos.

## Tecnologías usadas
- React
- JavaScript
- CSS
- Firebase / Firestore
- GitHub

## Objetivo
Cumplir con los requerimientos del capstone:  
Front end en React, CRUD de notas, persistencia en Firestore y documentación básica.
